Uso de simulación para fortalecer la educación en seguridad

Este proyecto explora cómo herramientas de modelamiento físico pueden ser utilizadas para enseñar y analizar escenarios de incendios y explosiones relevantes para Colombia.

Resumen del proyecto

El proyecto tiene como objetivo principal evaluar el uso de simulaciones como herramienta educativa en seguridad contra incendios y explosiones en Colombia. Para ello, se desarrolló una metodología que combina la evaluación de herramientas de modelamiento con la simulación de eventos reales o representativos.

Metodología

1. Evaluación de herramientas

Se desarrolló una rúbrica para evaluar herramientas como FDS, CFAST y Fluent, considerando criterios como costos, facilidad de uso y disponibilidad de información.

2. Selección de escenarios

Se identificaron eventos relevantes, preferiblemente en Colombia, relacionados con incendios y explosiones en distintos contextos.

3. Simulación

Se realizaron simulaciones de los eventos o de etapas críticas de los mismos, utilizando la herramienta más adecuada según la rúbrica.
